安装mysql

  1. check 系统是否安装或运行Mysql
  2. download mysql 源码包和解压

安装dbeaver

第一步：下载jdk windows 版本

Jdk下载页面

  • 不想下载目前最高版本可以拉到页面最下端， 点击java archive 即可找到历史版本

  • Java Archive

    The Java Archive offers access to some of our historical Java releases. WARNING: These older versions of the JRE and JDK are provided to help developers debug issues in older systems. They are not updated with the latest security patches and are not recommended for use in production.

安装jdk
配置环境变量
  1. 我的电脑-右键-属性-环境变量
  2. 点击下方的系统环境变量-新建
  3. 配置JAVA_HOME。新建，变量名JAVA_HOME，变量值，jdk路径，我的路径是D:\jdk1.8.0_202，保存。
  4. 配置CLASSPATH。新建，变量名CLASSPATH，变量值，.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（第一个分号前前面有一个点）
  5. 配置Path。打开Path变量，在变量值最前加入%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
第二步：下载dbeaver

官网：https://dbeaver.jkiss.org/download/

创建dbeaver快捷方式的时候要指定jdk的路径，快捷方式上右键 -> 属性 -> 快捷方式 -> 目标，编辑如下

-vm D:\jdk1.8.0_202\bin\javaw.exe
