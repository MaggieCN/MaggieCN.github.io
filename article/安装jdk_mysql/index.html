安装jdk （ 重点修改组、用户组）

安装
第一步：上传jdk文件从windows到linux

  • 上传命令：rz

  • 目前jdk的这个tar包是在根目录下，可以查看一下

  • 查看命令: ll

第二步：创建java文件夹，在/usr/路径下
  • 命令：mkdir -p /usr/java/
第三步：解压缩tar包到指定目录
  • 命令：tar -xzvf jdk-8u181-linux-x64.tar.gz -C /usr/java/
  • 在按到jdk的时候可以按一下tab键自动补全【新手请记住】
第四步： cd 到java路径下查看
  • 命令1： cd /usr/java
  • 命令2：ll

​ 可以看到已经产生了一个jdk-8u181的文件夹

第五步：更改用户，用户组
  • 命令： chown -R root:root jdk1.8.0_181

​ drwxr-xr-x 7 10 143 4096 Jul 7 2018 jdk1.8.0_181

  • 对应权限是755 用户 10 用户组143

  • d 代表文件夹

  • rwx 文件或者文件夹的所属用户有读写执行功能

  • r-x 文件或者文件夹的所属用户组的成员，只有读写功能

  • r-x 用户组的其他成员对这个文件或文件夹的权限，只有读写功能

更改后的： drwxr-xr-x 7 root root 4096 Jul 7 2018 jdk1.8.0_181

第六步： 配置环境变量

jdk是一个全局的环境变量

  • 命令：vi /etc/profile
  • 大写的G到最后一行
  • i 键 进入编辑模式
  • #Maggie java env
    export JAVA_HOME=/usr/java/jdk1.8.0_181
    export PATH=$JAVA_HOME/bin:$PATH
  • esc 键 再按 :wq 保存退出
第七步环境变量生效
  • 命令： source /etc/profile
第八步 检查是否安装成功/是否生效
  • 命令：java -version
  • 或者命令：which java
