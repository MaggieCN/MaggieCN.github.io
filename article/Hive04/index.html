PARTITIONED BY

概念

分区表：在关系型数据库中也有分区，分区中有操作日志表：

其主要作用：记录日志、查询操作——谁什么时间做了什么 who when what

以xxx_20201210.log 日期为单位存储

  • hive中的partition就是辅助查询，缩小查询范围
  • 加快数据的检索速度和对数据按照一定的规格和条件进行管理。
  • ＩＯ开销更小

创建分区表

desc formatted dept_partition

Partition Information

col_name data_type comment

day string

将几组分区数据放到data/partition/目录下

load到hive表中

元数据： PARTITIONS

PART_ID CREATE_TIME LAST_ACCESS_TIME PART_NAME SD_ID TBL_ID
1 1607955045 0 day=20201213 32 31
  • 如果直接把文件放到hdfs put命令存放，没有元数据的，所以不能载入到hive里，load data 可以刷新元数据，更新。使用：MSCK REPAIR TABLE dept_partition; 可以将元数据刷新进去【刷整个分区表，慎用】

【推荐】 增加分区

ALTER TABLE table_name ADD [IF NOT EXISTS] PARTITION ;

减少分区

ALTER TABLE dept_partition DROP PARTITION(day=‘20300503’),PARTITION(day=‘20300502’);

内置函数

显示所有 内置函数：show functions;

查看某种内置函数定义：desc function xxx;

【推荐】某内置函数&有例子：desc fucntion extended xxx;

date functions

  1. 当前日期： current_date 2020-12-12

  2. 当前时间戳： current_timestamp 2020-12-12 21:22:46.416

  3. 1970年到现在的一个时间戳： unix_timestamp: 1607779679

  4. 从时间戳转到想要的格式： from_unixtime（unix_time, format） “yyyyMMdd HHmmss”

  5. 只取年月日： to_date

  6. 只取年：year(), month day hour minute second

    weekofyear
    dayofmonth

    months_between

    这些返回都是Int

  7. 加月份：add_months

  8. 日期相差多少天：datediff

    例子：datediff(date1, date2)

  9. 加一天：date_add 例子：SELECT date_add(‘2009-07-30’, 1) FROM src LIMIT 1; 结果：‘2009-07-31’

    date_sub

  10. 当前月最后一天 ：last_day

    SELECT last_day(‘2009-01-12’) FROM src LIMIT 1;
    ‘2009-01-31’

数学
  1. 四舍五入：round —— round(x[, d]) - round x to d decimal places （5会进位）
  2. 取右值——ceil （向上取整）
  3. 取左值——floor
  4. 绝对值：abs
字符串

desc function extended split;

  1. 大小写转化：upper/lower

  2. 字符串长度：length

  3. 删除空格：trim

  4. 补齐（常用月份 5 补齐成05）lpad/rpad

  5. 正则替换：regexp_replace

  6. 取子字符串：substr

  7. 字符串拼接：concat

  8. 数组字符串拼接：concat_ws：concat_ws(separator, [string | array(string)]+)

    concat_ws(separator, [string | array(string)]+)

    ‘www.facebook.com’

  9. 字符串分割：split ——split(str, regex)

    注意’.'字符前要加转义

    错误 ：select split(“192.168.1.1”,".");

    正确： select split(“192.168.1.1”,"\\.");

  10. nvl (default, ‘如果default是null 替换的值’)

json
  1. json_tuple(jsonStr, p1, p2, …, pn) - like get_json_object, but it takes multiple names and return a tuple. All the input parameters and output column types are string.
  1. 拆分address地址： parse_url_tuple

SELECT parse_url_tuple(a.fullurl, ‘HOST’, ‘PATH’, ‘QUERY’, ‘REF’, ‘PROTOCOL’, ‘FILE’, ‘AUTHORITY’, ‘USERINFO’, ‘QUERY:k1’) as (ho, pa, qu, re, pr, fi, au, us, qk1) from src a;

例如

select parse_url_tuple(‘http://www.google.com/test/film?param1=value1&param2=value2’, ‘HOST’, ‘PATH’, ‘QUERY’);
